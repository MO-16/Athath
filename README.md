# دار الأثاث - Dar Al-Athath

فخامة وأصالة | Luxury and Authenticity

أثاث منزلي فخم يجمع بين الأصالة والمودرن بجودة لا تضاهى.

## نظرة عامة

مشروع تطبيق ويب حديث لعرض وبيع الأثاث المنزلي الفاخر، مبني باستخدام أحدث التقنيات.

## التقنيات المستخدمة

### Frontend
- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - كتابة الكود بأنواع بيانات قوية
- **Vite** - أداة البناء والتطوير السريع
- **Tailwind CSS 4** - تنسيق وتصميم الواجهات
- **Shadcn/ui** - مكونات واجهة مستخدم جاهزة
- **Radix UI** - مكونات يمكن الوصول إليها
- **Framer Motion** - حركات وانتقالات سلسة
- **React Hook Form** - إدارة النماذج
- **Zod** - التحقق من صحة البيانات
- **Lucide React** - أيقونات حديثة

### Backend
- **Express.js** - إطار عمل الخادم
- **Drizzle ORM** - التعامل مع قاعدة البيانات
- **PostgreSQL** - قاعدة البيانات
- **Passport.js** - المصادقة والتفويض
- **Express Session** - إدارة الجلسات

### أدوات التطوير
- **tsx** - تشغيل TypeScript مباشرة
- **drizzle-kit** - إدارة ترحيلات قاعدة البيانات
- **esbuild** - بناء سريع

## متطلبات التشغيل

- Node.js 20+
- PostgreSQL
- npm أو pnpm

## التثبيت

```bash
# تثبيت المكتبات
npm install

# إعداد متغيرات البيئة
# تأكد من وجود DATABASE_URL في ملف .env
```

## أوامر التطوير

```bash
# تشغيل خادم التطوير
npm run dev

# تشغيل عميل التطوير فقط
npm run dev:client

# التحقق من الأنواع
npm run check

# دفع تغييرات قاعدة البيانات
npm run db:push
```

## البناء والإنتاج

```bash
# بناء المشروع للإنتاج
npm run build

# تشغيل الخادم في وضع الإنتاج
npm run start
```

## بنية المشروع

```
├── client/          # كود الواجهة الأمامية
│   └── src/         # ملفات المصدر
├── shared/          # الكود المشترك (الأنواع، المخططات)
├── server/          # كود الخادم
├── migrations/      # ترحيلات قاعدة البيانات
├── attached_assets/ # الأصول والملفات المرفقة
└── dist/            # ملفات الإنتاج المبنية
```
